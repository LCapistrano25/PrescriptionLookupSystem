# NaTampa - Gerenciador de Receitas

NaTampa é uma aplicação web moderna para busca, visualização e gerenciamento de receitas culinárias. O projeto integra uma interface intuitiva com uma API robusta para fornecer uma experiência completa aos usuários interessados em culinária.

## 🚀 Funcionalidades

- **Autenticação de Usuário**: Sistema de login seguro para acesso a funcionalidades personalizadas.
- **Exploração de Receitas**: Visualização de uma ampla variedade de receitas com busca em tempo real.
- **Detalhes da Receita**: Informações completas sobre ingredientes, modo de preparo e tempos de cozimento.
- **Favoritos**: Possibilidade de salvar suas receitas preferidas para acesso rápido.
- **Receitas da Semana**: Planejamento semanal de refeições integrado à plataforma.
- **Interface Responsiva**: Design adaptável para diferentes tamanhos de tela (desktop e mobile).

## 🛠️ Tecnologias Utilizadas

- **Frontend**:
  - HTML5 & CSS3
  - JavaScript (ES6+)
  - [Font Awesome](https://fontawesome.com/) para ícones
- **Backend**:
  - API REST desenvolvida em Python (Django/Flask)
  - Banco de dados remoto

## 📂 Estrutura do Projeto

- `/html`: Contém as páginas da aplicação.
- `/css`: Estilos organizados por componente/página.
- `/javascript`: Lógica de interação e integração com a API.
